# Power BI Sales Dashboard (Portfolio Project)

## Project Summary

This project demonstrates my ability to take raw, unstructured datasets and turn them into a **clean, interactive, and insight-driven Power BI dashboard**.

Using **Products, Regions, and Sales datasets**, I built a solution that answers key business questions around:

* Revenue performance
* Regional trends
* Product-level insights

The focus wasn’t just building visuals—it was about **creating a tool that supports decision-making**.

## Business Problem

Organizations often struggle to:

* Track sales performance across regions
* Identify top-performing products
* Detect trends and growth opportunities

This dashboard solves that by providing a **single source of truth** with clear, interactive insights.

## What This Project Demonstrates

This is not just a dashboard—it reflects core data analyst skills:

### Data Cleaning & Transformation

* Handled missing and inconsistent values
* Standardized formats across datasets
* Ensured data integrity before analysis

### Data Modeling

* Built relationships between **Products, Regions, and Sales tables**

These measures allow deeper analysis beyond basic aggregation.

## Dashboard Features

### Interactive Visuals

* Sales trends over time (line charts)
* Regional performance comparisons (bar/column charts)
* Product distribution (pie charts)
* KPI indicators for quick performance checks

### AI-Powered Insights

* Automated summaries highlighting key trends
* Quick detection of patterns and anomalies

### User Experience

* Clean, intuitive layout
* Dynamic filters and slicers
* Drill-down functionality for deeper exploration

## Key Insights Delivered
